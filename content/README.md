# 문서 작성 스타일

지향하고자 하는 바는 위키 형식의 지식 보관소. wikipedia 나 namuwiki 같은 일반적인 위키 페이지들을 생각하면 된다.

이전 obsidian 작성시엔 todo 를 달아놓고 유기되는 문서들이 너무 많았다. 이번엔 todo 를 많이 쓰지 말고 최대한 깊이 우선 작성을 해보자.
